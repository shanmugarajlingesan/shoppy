# shoppy (Next.js static demo)

A simple static storefront built with Next.js App Router.

## Features
- Static catalog with many products (Men/Women/Kids/Beauty)
- Search + filtering (brand, price range, category on All page)
- Sorting (price/rating/discount)
- Pagination (9 products per page)
- Product detail + client-side cart (localStorage)

## Run locally
```bash
npm install
npm run dev
```
